## 文章优化方向

 - 健壮
    - 阅读相关文章, 完善文章内容
 - 可读
    - 增加动画, 图解, 说明等提高可读性
 - 生态
    - 相关的知识链接, 对应的代码地址与视频讲解
 - 完整
    - 每个知识点都需要从以下几个方面进行讲解, 深入浅出
        - 是什么
        - 有什么用
        - 怎么使用(实现, 刷题)
        - 怎么用好(优缺点、应用场景)
        - 为什么这么用(底层原理, 设计原理, 启发)
